# CreatePicture

一个简单易用的文字图片生成器，允许用户自定义文字内容、样式和图片属性，实时预览并下载生成的图片。

## 功能特性

### 🎨 自定义样式
- **文字内容**：支持多行文本输入
- **文字大小**：可调整字体大小（6-100像素）
- **字体选择**：多种字体可选，包括常用中文字体
- **文字颜色**：丰富的预设颜色选项
- **背景颜色**：支持透明背景和多种预设颜色

### 📐 图片设置
- **图片尺寸**：自定义宽度和高度（100-2000像素）
- **圆角设置**：可调整图片圆角半径（0-100像素）
- **内边距**：精确控制文字与图片边缘的距离

### 🎯 实时预览
- 即时查看修改效果
- 响应式设计，适配不同屏幕尺寸

### 💾 便捷操作
- **本地存储**：自动保存用户配置，下次访问时恢复
- **一键下载**：生成PNG格式图片并快速下载

## 快速开始

### 在线使用
直接打开 `CreatePicture.html` 文件即可使用，无需安装任何依赖。

### 本地部署
1. 克隆或下载项目到本地
2. 使用浏览器打开 `CreatePicture.html` 文件
3. 开始创建您的文字图片

## 使用方法

1. 在**文字内容**输入框中输入您想要显示的文字
2. 调整**文字大小**、**图片尺寸**、**圆角半径**和**内边距**
3. 从**字体选择**中选择合适的字体
4. 选择**背景颜色**和**文字颜色**
5. 实时查看**预览效果**
6. 点击**下载图片**按钮获取生成的PNG图片

## 技术实现

### 前端技术栈
- **HTML5**：页面结构和语义化标签
- **CSS3**：响应式布局和样式设计
- **JavaScript**：交互逻辑和图片生成
- **Tailwind CSS**：实用优先的CSS框架
- **Font Awesome**：图标库

### 核心功能实现
- **Canvas API**：用于绘制和生成图片
- **Local Storage**：保存用户配置
- **Blob API**：图片下载功能
- **响应式设计**：适配不同屏幕尺寸
